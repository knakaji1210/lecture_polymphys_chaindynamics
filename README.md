# lecture_polymphys_chaindynamics

講義「高分子物理学特論」の#7「Dynamics of a Single Chain」で利用  

格子モデル（理想鎖の場合）が緩和時間と拡散係数について、Rouseモデルと同じスケーリング則を与えることを観測することができた面白い試み  

＜2601のアップデート＞  
SAW鎖を実装  
これによって、逆に色々なミスが見つかり、結果として  
各ステップごとで、配置を変更できるところだけが変更されるようにプログラムをアップデートできた  
その際、リストの複製する際に  
new_list = list.copy()  
としないといけないことを完全に忘れていた・・・
この変更によって、理想鎖はv2からv3へ、SAW鎖はv1のまま、FH鎖は手をつけてない
