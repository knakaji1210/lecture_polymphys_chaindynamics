# lecture_polymphys_chaindynamics

講義「高分子物理学特論」の#7「Dynamics of a Single Chain」で利用

格子モデル（理想鎖の場合）が緩和時間と拡散係数について、Rouseモデルと同じスケーリング則を与えることを観測することができた面白い試み

＜2601のアップデート＞
SAW鎖を実装
各ステップごとに、配置を変更できるところだけが変更されるようにプログラムをアップデート
要は、updated_coordinate_listを作り、変更はそこに入れ、coordinate_listは前のステップのままにしておき、
最後にupdated_coordinate_listをcoordinate_listに更新する形にした
この変更によって、理想鎖はv2からv3へ、SAW鎖はv1のまま

＜未解決問題＞ 
Flory-Huggins鎖の実装は不十分、末端だけは実装できているが、どうやって平均場を実装できるかが今後の課題
ステップ数を大きくしたときのムービーgifについて、データを間引く必要もある（多分、これはここだけの問題じゃないが・・・）
配置を行列に置き換えて計算できないだろうか、が、鎖の繋がりをどう数値化するかのハードルがある
拡散係数については、次元性に関連する係数がうまくいってないかどうか確認する必要がある
